# ⚡ VORTEX
> Personal device control system — Control your PC & Laptop from your Android phone

## Features
- 📡 Auto WiFi scan — device list automatically
- 🖥️ Live screen streaming + touch control
- 💻 Shell terminal
- 🎵 Media control (Spotify/YouTube/VLC)
- 🔔 Smart alerts (battery/USB/internet/crash)
- 📋 Clipboard auto-sync
- 📁 File browser + transfer
- 🖨️ Remote print
- 📦 App manager (install/uninstall)
- 💬 Device-to-device messages
- ⚙️ Automation scheduler

## Setup
1. Run `VortexAgent.exe` on PC
2. Set password in setup window
3. Install APK on phone
4. Scan → Connect → Done!
